Software Engineer

Jakarta, Jabodetabek Waktu Penuh Rp 12.000.000 - Rp 15.000.000

Deskripsi Pekerjaan

Software Engineer adalah seorang profesional TI yang merancang, mengembangkan, dan memelihara perangkat lunak komputer di sebuah perusahaan. Mereka menggunakan kreativitas dan keterampilan teknis mereka dan menerapkan prinsip-prinsip rekayasa perangkat lunak untuk membantu memecahkan masalah baru dan berkelanjutan untuk suatu organisasi.

Tanggung Jawab

  • Analisis kebutuhan pengguna
  • Tulis dan uji kode, perbaiki dan tulis ulang seperlunya dan komunikasikan dengan programmer mana pun yang terlibat dalam proyek
  • Up to date teknologi baru
  • Mengembangkan software

Kualifikasi

  • 4 Tahun Pengalaman dalam Programming Language(.NET MVC/Core, Java Springboot, PHP)
  • Familiar dengan GIT
  • Familiar dengan SQL
  • Menguasai OOP concept
  • Familiar dengan Framework(Vue.JS, React.JS, Angular.JS, dll.)
